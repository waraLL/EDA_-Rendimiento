# EDA - Rendimiento Académico 
Análisis exploratorio de datos (EDA) sobre variables asociadas al rendimiento académico y abandono estudiantil

## Objetivo
Realizar un análisis exploratorio de datos para identificar patrones relacionados con el rendimiento académico y abandono estudiantil.

## Herramientas utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Procesos realizados
- Limpieza de datos
- Tratamiento de nulos
- Detección de outliers
- Análisis exploratorio
- Visualización de datos
